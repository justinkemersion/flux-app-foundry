# Generated documentation

Files in this directory are produced by:

```bash
pnpm foundry:report
```

| File | Contents |
|------|----------|
| `routes.md` | App Router route inventory |
| `components.md` | Component inventory with client/server flag |

Reports with enforcement detail live in `.local/reports/` (gitignored).

Regenerate after structural changes so Cursor and reviewers share current architecture context.
